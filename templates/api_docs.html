API Dokumentace

Autentizace

Všechny API požadavky vyžadují Bearer token v hlavičce Authorization. 

Authorization: Bearer <váš-token>

Tokeny: Read (jen čtení) nebo Read & Write (čtení i zápis).

Sady slovíček

GET/api/setsREAD
Seznam všech sad.
POST/api/setsRW
Vytvoří novou sadu.
Tělo:{"name": "Lekce 2", "lang_a": "cs", "lang_b": "es"}
GET/api/sets/:idREAD
Detail sady včetně slovíček.
PUT/api/sets/:idRW
Upraví sadu (name, lang_a, lang_b).
DELETE/api/sets/:idRW
Smaže sadu a všechna slovíčka.
POST/api/sets/:id/shareRW
Zapne/vypne sdílení (toggle). Vrátí share_url.

Slovíčka

GET/api/sets/:id/wordsREAD
Seznam slovíček v sadě.
Odpověď: 
[{"id": 1, "word_a": "DECIR", "word_b": "říkat"}, ...]
POST/api/sets/:id/wordsRW
Přidá slovíčko.
Tělo:{"word_a": "DECIR", "word_b": "říkat"}
PUT/api/words/:idRW
Upraví slovíčko.
DELETE/api/words/:idRW
Smaže slovíčko.

Hromadný import

POST/api/sets/:id/importRW
Hromadný import — JSON pole nebo textový formát.
Varianta 1 — JSON: 
{"words": [{"word_a": "DECIR", "word_b": "říkat"}, ...]}
Varianta 2 — text (středník/tabulátor): 
{"text": "DECIR;říkat\nDAR;dát\nIR;jít/jet"}
